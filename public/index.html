Local CLI First

本地 CLI 主流程

  1. 首次只需要执行一次 `client/bootstrap.sh` 或 `client/bootstrap.ps1`。
  2. 填好 `client/config.env` 后，在当前设备当前网络下直接执行 `subup`。
  3. 脚本会在本机测速并把最新 Top200 推送到 Worker 固定订阅。
  4. 回到 Clash / v2ray / Shadowrocket / Surge 点击“更新订阅”。
